# Thai Grapheme-to-Phoneme
Thai Grapheme-to-Phoneme (Thai G2P)

I using data from http://thai-notes.com/ and ICU.

## Install

install pythainlp from github

```
pip install https://github.com/PyThaiNLP/pythainlp/archive/dev.zip
```

and install pyicu.

## Using

run

```
python thai2ipa.py
```
